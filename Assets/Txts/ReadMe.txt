현재 보이는 문제점

카메라의 부자연스러움
	-> 시네머신 카메라로 목표하는 느낌의 카메라 무브먼트 구현.
	-> 또는 에셋 플러그인을 활용하여 목표하는 느낌의 카메라 무브먼트 구현.

연속킬 시 카메라 이벤트
	-> 총기 연속 사살 시에는 적용 X 레이캐스트로 먼거리까지 쏠 수 있기에, 연속 사살 애니메이션이
		총기에도 적용 된다면, 어지러움을 유발할 것임.
	-> fmod를 활용해서, bgm의 재생 속도를 0.2배로 감소, 슬로우 모션 0.2배 보간을 활용한 카메라 1.5배 확대 후,
		일정 시간이 지난 뒤, 원래 화면으로 복구.
	-> 적용되는 무기: 근접 무기, 샷건 (레이케스트를 통해 샷건의 거리를 제한할 것임.)
